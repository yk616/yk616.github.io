前言

这份学习笔记整理自课上学习的笔记以及matplotlib上提供的样例。尽量简明扼要。

matplotlib是一套面向对象的绘图库，绘制的图标中每个绘图元素（例如线条、文字、刻度等）均为对象。使用matplotlib绘图，主要使用了其中的pyplot工具包。

可以在控制台中输入python -i或ipython，进入python交互界面，然后进行matplotlib绘图。

关于annotate方法内部的参数，简述如下：

  • xycoords：指定xy的坐标形式，默认为与图像相同的坐标（data），无需特别设置

  • textcoords：指定xytest的坐标形式，默认为与图像相同的坐标，可以设置为offset points，相对xy的坐标

  • fontsize：指定文字内容的大小（像素）

子图

matplotlib绘制子图有多种方法，例如：axes、subplot、subplots、GridSpec等。下面介绍前三种。

  • subplot

    使用subplot方法需要三个参数指定特定图表，分别是(行数, 列数, 序数)。例如，subplot(2, 2, 3)即指的是生成2行2列，从左到右。从上到下第3幅图，即是4幅图中左下角的那幅。在这一行后可以输入代码对这幅图进行调整。

    可以在subplot内添加facecolor

  • subplots

    使用subplots，返回两个元素，分别是画布、子图。因此，需要定义两个变量接收。例如，设定子图的变量为ax，则ax[1, 0]（或ax[1][0]）即表示第2行第1列的图表。
