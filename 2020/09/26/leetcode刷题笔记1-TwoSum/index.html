双指针算法

双指针算法是一种遍历有序对象的方式。通常情况下对对象的遍历是从头开始每一项访问，这就是单指针。双指针指的是同时有两个指针去遍历对象，从而节省时间的算法。一般情况下，双指针的时间复杂度为O(n)O(n)O(n)，空间复杂度为O(1)O(1)O(1)。该算法分为两类：

（1）左右指针

定义两个指针，把指向有序数组第一个元素的指针称为左指针，指向最后一个元素的指针称为右指针。其伪代码如下：

C++的sort函数

C++中有默认的排序函数，即sort函数，用于对容器或普通数组中的元素进行排序，默认为升序。其时间复杂度为O(nlog⁡n)O(n\log n)O(nlogn)，空间复杂度为O(log⁡n)O(\log n)O(logn)。

使用方式：

  1. #include <algorithm>

    引入头文件

  2. using std::sort;

    注意模板类

  3. 语法

    sort(first, last, comp);

  • first和last：排序范围的起始位置和结束位置
  • comp：排序规则，其中标准库提供的一些规则有：
    std::greater<int>，升序排序（默认）
    std::greater<int>，降序排序

示例如下（该示例来自 https://www.cnblogs.com/junbaobei/p/10776066.html ）：

需要指出的是，sort函数采用的是快速排序算法，而该算法是不稳定的。想要使用稳定的算法，应使用stable_sort函数。更多的内容可以参考：C++ sort()排序函数用法详解。

C++的map容器

C++的map容器有4种，这里先记录map<K, T>，K指的是键，T指的是对象（值），键不允许有重复，但是值允许。map中的键是有序的，能够根据键值快速查找对象，查询的时间复杂度是O(log⁡n)O(\log n)O(logn)，增加和删除的时间复杂度是O(log⁡n)O(\log n)O(logn)。

（1）创建

  1. #include <map>

    引用头文件

  2. use std::::map;

    注意模板类

  3. 一些创建方式：

    • map<int, std::string> mp;

      创建一个名称为mp的map，键类型为int，值类型为string

    • mp[0] = "spring";

      直接赋值法：添加一个键值对0-spring，可以只创建键而不赋值

    • mp.insert({1, "summer"});

      insert方法：添加一个键值对1-summer

    • mp = {{2, "autumn"}};

      列表初始化：直接赋一个键值对2-autumn给map，此法适用于C++11及以上版本

  4. map<int, std::string>::iterator mp_iter;

    定义一个名称为mp_iter的iterator

（2）一些简单的方法

  1. mp[0];

    获得0这个键对应的值

  2. mp.size();

    获得map所包含的元素数

  3. 遍历：该代码是定义一个名称为mp_iter的iterator，依次输出pap内的每个键值对，其中mp_iter->first指的是键，mp_iter->second指的是值。

  4. mp1.swap(mp2);

    交换两个map，mp1和mp2

  5. 查找：

    • mp.count(0);

      查找某个键是否存在，是返回1，否返回0

    • mp_iter = mp.find(0); if (mp_iter != mp.end()) cout << mp_iter->second << endl;

      查找某个键是否存在，是则可以通过mp_iter获得该键值对，否返回mp.end()*。

  • 查找某个值是否存在，存在则输出相应的键，由于允许值重复，故可能输出多个键

  1. 删除：

    • mp.erase(0);

      删除键为0的键值对

    • mp.erase(mp.begin(), mp.end());

      清空整个map

    • mp.clear();

      效果同上
