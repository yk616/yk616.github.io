自己写的将英文文段中的拉丁字母进行替换的脚本。

前言

最近在浏览知乎时发现了一个这样的有趣的问题：

有没有哪种语言只要稍微改动一下写法就能让翻译引擎无法识别但是掌握该门语言的人阅读起来还是没有太多障碍？

然后受到启发，产生了做一个类似工具的念头。

正题

字母替换

这个工具是将英文文段中的各个字母进行替换，然后生成替换后的文段。

下面是两个文本框，第一个文本框输入需要处理的文段，第二个文本框输入从A到Z，从a到z依顺序替换每个字母的符号，限定输入52个符号，中间可以有空格。

点击“替换”可以将原文段进行替换，点击“还原”可以将进行替换后的文段还原。

请输入需要进行替换的内容。



这种替换方式的本质是单表代换，详见维基百科。

伪西里尔字母

维基百科对伪西里尔字母的介绍：

伪西里尔字母（英语：faux Cyrillic、pseudo-Cyrillic），或称伪俄文（faux Russian、pseudo-Russian）是替代外观上相似的拉丁字母的西里尔字母，以让人想到这个词与俄罗斯、苏联、东欧有关。

下面是自己做的将英文文段转变成这种字母的工具。

替换规则有：
A -> Д; G -> ☭; IO -> Ю; N -> И; R -> Я; U -> Ц; W -> Ш; X -> Ж; Y -> У


eu4编码转换器

看到了paratranz上的MOD 工具，于是就想拿到自己的博客试一试。脚本来源自：https://gist.github.com/bruceCzK

此工具只适用于新版的eu4（1.26版本之后）的编码，将编码从cp1252转换至utf8。
