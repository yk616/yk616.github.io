树莓派4B入手配置

发表于 2019-12-02 分类于 求知 阅读次数： Valine：
本文字数： 5.3k 阅读时长 ≈ 5 分钟

背景

前些时候入手了今年新出的树莓派4B，系统配置为2GB版本加16GSD卡。其实如果资金充裕的话，还是应当购买4GB版本加32GSD卡使用会更加流畅。其余配件包括：官方充电线（带开关，5V3A），风扇，（散热片，）读卡器，外壳（建议购买能够方便向引脚接线的外壳）。没有额外购买显示屏。合计支出在550元左右。但是在之后的配置过程中，因为在网上所查阅的资料质量良莠不齐，走了许多弯路。在此将自己整理好的配置教程放在这里。

配置过程

一、镜像烧录

树莓派4B的CPU架构为arm7l（不是arm64，需要在以后的软件安装中留意）。
树莓派可以安装许多种类相对应的操作系统，除官方镜像外，在GitHub上可以找到适配树莓派4B的64位操作系统镜像。作为入门小白，选择安装官网上提供的基于Debian10（buster）的32位raspbian系统镜像（大小约6G），包含了常用的功能。
此外需要下载安装的软件有：

  • SD卡格式化软件 SD Card Formatter
  • 系统镜像烧录软件 win32diskimager
  • 远程控制软件 VNC Viewer

将SD卡用格式化软件点format进行格式化。格式化之后用烧录软件选择下载好的镜像以及SD卡之后点write进行写入。
因为我未购买显示屏，所以需要使用远程连接方式控制树莓派。
用记事本创建两个文件，第一个命名为ssh，不带后缀；另一个文件输入能使树莓派连接的WiFi信息（校园网等公共网络无法连接，可用手机热点代替），格式如下（参考：http://shumeipai.nxez.com/2017/09/13/raspberry-pi-network-configuration-before-boot.html ）：

其中，ssid为WiFi的名称，psk为该WiFi密码，priority为该WiFi连接的优先级（数字越大越优先连接）。确保树莓派连接的WiFi是自己的电脑也在同时连接的。
文件保存后命名为wpa_supplicant.conf，用conf替代原来的txt后缀。将两个文件直接放在烧录完镜像之后的SD卡的根目录下。
打开Windows命令行，输入arp -a查看所有已连接设备的IP地址，记下。
完成上述工作之后，将TF卡插入树莓派，开机。

二、远程连接树莓派

确认树莓派的IP地址。
待树莓派启动后几分钟，若上一步的WiFi设置无误，即树莓派已经同电脑连接在同一个WiFi之下后，再次打开Windows命令行，输入arp -a查看所有已连接设备的IP地址，观察相较于树莓派未开机前多了哪一个，并确保其为：

  • 静态
  • IP地址最后一个数字不是1或者255。

该多出来的IP地址即为树莓派的IP地址。
（查找IP地址的其他方法例如可以使用软件“ipscan22.exe”，使用方式参考：https://www.cnblogs.com/SystemCall/archive/2019/06/16/11032391.html ）

输入sudo raspi-config进入树莓派系统配置。大概的界面如下：
接下来要配置的有：默认时间、默认地区、默认语言、默认开机方式。
这里使用方向键↑↓进行选择，方向键←→在显示选择和返回时使用，按回车键确认，空格键用于某些时候的选择。
其中需要修改如下所示的几个选项。
1、Change user password。前面若没有修改可以在这里修改。
2、Boot options。选择B1，再选择B4，目的是使之后开机默认进入图形界面。选择B3，再选择no，关闭开机前动画。
3、Localisation options。选择I1系统调整语言，会有多种语言的选项，按空格键进行选择，被选择显示的语言会在左边[ ]内有*号。其中有一些是必须选择的：

  • en_GB.UTF-8、UTF-8、zh_CN.UTF-8
    选择完之后按回车键，接着选择默认语言，在其中选择zh_CN.UTF-8。等待一段时间。
    接着选I2调整时区。选择Asia之后选择shanghai。
    接着选I4调整网络地区。选择CN China。
    4、Interfacing options。选择开启P1 camera、P2 SSH、P3 VNC、P6 serial、P8 remote gpio。
    5、Advanced options。开启A1，用于充分利用TF卡空间。开启A2，在整个屏幕上显示。选择A4，再选择force 3.5mm，使声音从耳机孔流出。设置分辨率，选择A5，再选择DMT mode 85或者更大的分辨率，才能够在使用vnc时正常显示，否则会显示cannot currently show the desktop。选择A6，提高显示画质（但是在4B上未配置成功）。
    6、Update。可以选择但是一般不会更新成功（涉及到换源问题）。
    （参考：http://www.waveshare.net/study/article-597-1.html ，但是此教程对应的树莓派版本较旧）
    设置完毕，选择yes进行重启。

三、进入图形界面

现在可以使用ssh或者vnc进入树莓派图形界面。

1、ssh

在开始菜单→Windows附件中找到远程桌面，在“计算机”中输入树莓派的IP地址，进入登录界面之后输入用户名pi和你设定的密码就可以进入树莓派桌面。
除了使用远程桌面连接ssh，还可使用其他软件进行登陆，例如putty、xshell6等。

2、vnc

打开已经安装好的vnc viewer，输入树莓派IP地址，点continue（第一次连接），之后输入用户名pi，密码（可以选择下面的remember password，以后不用再输入），进入树莓派桌面。
进入图形界面时的操作速度取决于：（1）树莓派本身内存大小；（2）运行远程桌面机器处理器。例如，树莓派4B的2G版本下，系统操作速度：ssh（电脑端）>vnc（电脑端）>vnc（iOS）端。
不建议同时登录ssh和vnc。
第一次进入桌面会自动提示进行配置初始设定，其中第一个点“use us keyboard”之后选择next，第二个密码进行修改后next（必须输入内容，可以将你设定好的密码再输进去就可），第三个可以直接next，第四个WiFi界面直接skip，完成配置（因为前面已经配置WiFi了）。
如果你一直点next，会跳到系统update software界面进行更新，但是一般会失败，换源问题。
（其他可以vnc连接的软件还有更强大的teamviewer，配置方式参考：
https://github.com/TommyZihao/Zihao-Blog/blob/master/02一劳永逸配置树莓派.md ）
